Esta carpeta almacena los gráficos y pantallazos generados por el sistema.

Los gráficos se guardan automáticamente cuando:
- Generas un gráfico desde el análisis técnico
- El sistema guarda una captura de una señal
- Guardas manualmente un gráfico desde la interfaz

Formato de archivos: PNG
Nomenclatura: TICKER_YYYYMMDD_HHMMSS.png
